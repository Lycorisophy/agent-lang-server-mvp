## v0.6→v1.0 路线图（调整后）

| 版本 | 主题 | 核心交付 |
|------|------|----------|
| **v0.6** | **Agent 地基 + 常用工具 + 提示词工程** | 统一编排框架骨架、工具注册中心（含知识库查询/提取/事件查询/搜索）、提示词模板管理与核心模板调优 |
| **v0.7** | **ReAct 模式** | 实现 Reason-Act-Observe 循环，增加显式思考节点，优化工具调用与停止条件 |
| **v0.8** | **Plan & Execute 模式** | 计划生成与执行分离，支持步骤依赖、重试和重规划 |
| **v0.9** | **SubAgent 模式** | 主 Agent 可调度子 Agent，实现多智能体协作基本框架 |
| **v0.10** | **Dify 工作流集成** | 将 Dify 工作流作为工具接入，支持同步/流式调用 |
| **v0.11** | **对话状态追踪 (DST)** | 提取对话槽位、意图，维护对话状态并影响工具选择 |
| **v0.12** | **定时任务 Tool** | 实现定时任务创建、管理、执行工具 |
| **v0.13** | **自动技能工匠** | 自动生成技能脚本、测试、注册到 SkillHub |
| **v0.14** | **Ask & Auto 模式** | 提供操作前确认（Ask）与全自动（Auto）两种交互模式 |
| **v0.15** | **降级策略与全链路执行日志** | 模型/工具调用超时重试、熔断、多模型故障转移；详细的 `agent_execution_log`，支持成本统计与性能分析 |
| **v1.0** | **安全加固 + 性能优化** | 全面安全审计、最小权限、敏感信息脱敏、沙箱强化；缓存优化、数据库索引调优、大规模压测，发布稳定版 |
